# Custom-BI-Publisher-Report-Supplier-Invoice-Summary
1.Created a BI Publisher report to show supplier invoice details.
2.Used SQL to join invoice and supplier tables and fetch required data.
3.Built Data Model to organize the data for the report.
4.Designed report layout using RTF template for clear output.
5.Implemented bursting to send reports automatically to users in PDF format.
6.Scheduled the report using ESS Job to run automatically.
